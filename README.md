This project contains an automated Python tool that selects the optimal bearings for the input shaft of a mechanical assembly.
The script computes loads, life, safety factors, and geometric constraints to help designers choose bearings efficiently and accurately.

The repository also includes CAD files related to the input shaft assembly and overall electromechanical design.
